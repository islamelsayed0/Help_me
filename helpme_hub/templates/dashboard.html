Common Issues – Quick Tickets

Try these short steps first. If it still is not working, use the button to open a ticket in a few seconds.

Printer help

  1. Make sure the printer is powered on and has paper.
  2. Check you selected the correct printer on your computer.
  3. Try printing a simple one‑page test document.
  4. If it still will not print, open a ticket below.
Open printer ticket

Account / login help

  1. Check you are using the correct username or email address.
  2. Try the “forgot password” option if it is available.
  3. Close the browser, reopen it, and try signing in again.
  4. If you still cannot sign in, open a ticket below.
Open account ticket

Device / Promethean board

  1. Confirm the device and cables are firmly plugged in and powered on.
  2. Check the input/source on the screen or projector.
  3. Turn the device off, wait ten seconds, then turn it back on.
  4. If it is still not working, open a ticket below.
Open device ticket
